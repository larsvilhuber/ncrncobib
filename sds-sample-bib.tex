This bibliography serves as an example how to (a) cite data (b) cite the relevant documentation for the data present on the  Synthetic Data Server. The bib files are pulled directly from the web server at \url{https://www2.vrdc.cornell.edu/}. Articles are in alphabetical order of the first author's last name.

For instance, to cite the SIPP Synthetic Beta version 6.0 data, you would use \parencite{SSB602}, and in addition cite its documentation \parencite{CED2AR-SSBv602}. You might also add the document describing its creation \parencite{CreationSSB}.
